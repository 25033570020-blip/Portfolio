# Khushi's Portfolio Website

A responsive personal portfolio website built using HTML, CSS, and JavaScript to showcase my skills, education, internship experience, and web development projects.

## Features

- Responsive design for desktop and mobile devices
- About Me section
- Education details
- Internship experience
- Skills showcase
- Project portfolio with GitHub and Live Demo links
- Contact information
- Resume download option

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Git & GitHub
- Vercel (Deployment)

A modern coffee shop landing page featuring responsive design, branding, and user-friendly layouts.

A web application that calculates a user's exact age in years, months, and days using JavaScript date functions.
